# Wiki-Path-Finder
This code find the shortest path between two articles<br />
Libraries Used -<br />
xml.etree.ElemenTree(For parsing XML dumps of Wikipedia)<br />
Mwparserfromhell (For extracting the links from Xml)<br />
Tkinter For Basic GUI<br />
