# Wiki-Path-Finder
This code find the shortest path between two articles
Libraries Used -
xml.etree.ElemenTree(For parsing XML dumps of Wikipedia)
Mwparserfromhell (For extracting the links from Xml)
Tkinter For Basic GUI
